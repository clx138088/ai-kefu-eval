## 字段说明

| 字段              | 含义                          |
| --------------- | --------------------------- |
| `case_id`       | 评测 case 编号                  |
| `user_query`    | 用户原始提问                      |
| `预期意图`          | 人工标注的意图                     |
| `ai_answer摘要`   | AI 实际回答内容概括                 |
| `显性诉求`          | query 中用户说出口的诉求             |
| `隐含诉求`          | 用户没说但指向的真实需求                |
| `分类`            | A / B / C / D / E           |
| `等级`            | 1/2/3（A/B/C类）；1/2（D类）；—（E类） |
| `decision_path` | 触发的步骤 + 判断                  |
| `cascade`       | A 类标记"是"，其余"—"              |
